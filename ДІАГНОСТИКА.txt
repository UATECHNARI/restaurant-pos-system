┌──────────────────────────────────────────────────────────────────┐
│  КРОК 1: Відкрити DevTools                                       │
└──────────────────────────────────────────────────────────────────┘

  1. Відкрийте браузер з системою (http://localhost:5173)
  2. Натисніть F12 (або Ctrl+Shift+I / Cmd+Option+I)
  3. Перейдіть на вкладку "Console"


┌──────────────────────────────────────────────────────────────────┐
│  КРОК 2: Перезавантажити сторінку                                │
└──────────────────────────────────────────────────────────────────┘

  1. Натисніть F5 або Ctrl+R (Cmd+R на Mac)
  2. Зачекайте поки сторінка завантажиться
  3. Подивіться на Console в DevTools


┌──────────────────────────────────────────────────────────────────┐
│  КРОК 3: Перевірити помилки                                      │
└──────────────────────────────────────────────────────────────────┘

  Чи є ЧЕРВОНІ помилки в Console?
  
  ✅ Якщо НІ - перейдіть до Кроку 4
  
  ❌ Якщо ТАК - скопіюйте текст помилки і відправте мені
     Приклад помилок:
     • "Cannot read property 'map' of undefined"
     • "Uncaught TypeError: ..."
     • "404 Not Found"
     • "Network Error"


┌──────────────────────────────────────────────────────────────────┐
│  КРОК 4: Перевірити Network                                      │
└──────────────────────────────────────────────────────────────────┘

  1. Перейдіть на вкладку "Network" в DevTools
  2. Перезавантажте сторінку (F5)
  3. Подивіться чи всі запити успішні (зелені, статус 200)
  
  Чи є ЧЕРВОНІ запити (помилки 404, 500)?
  
  ❌ Якщо ТАК - скопіюйте URL помилкового запиту


┌──────────────────────────────────────────────────────────────────┐
│  КРОК 5: Спробувати натиснути кнопку                             │
└──────────────────────────────────────────────────────────────────┘

  1. Спробуйте натиснути на одну з кнопок ролей
  2. Спробуйте ввести текст в поле Email
  3. Чи щось відбувається?
  
  ЩО САМЕ ВІДБУВАЄТЬСЯ:
  □ Нічого не відбувається
  □ Кнопка натискається, але є помилка в Console
  □ Поле вводу не дозволяє вводити текст
  □ Все виглядає "сірим" або напівпрозорим
  □ Інше: ___________________________


┌──────────────────────────────────────────────────────────────────┐
│  КРОК 6: Перевірити backend                                      │
└──────────────────────────────────────────────────────────────────┘

  Відкрийте в браузері:
  http://localhost:3001/health
  
  ЩО ВИ БАЧИТЕ:
  ✅ {"status":"OK","timestamp":"..."} - Backend працює
  ❌ "Сайт не знайдено" - Backend не запущений

  КРОК 1: Зупинити backend і frontend
  ────────────────────────────────────
  • Закрийте ВСІ вікна PowerShell з запущеними серверами
  • Або натисніть Ctrl+C в кожному вікні


  КРОК 2: Очистити кеш браузера
  ──────────────────────────────
  • В браузері натисніть Ctrl+Shift+Delete
  • Виберіть "Кешовані зображення та файли"
  • Натисніть "Очистити дані"


  КРОК 3: Запустити знову
  ────────────────────────
  
  PowerShell #1 (Backend):
  cd D:\Work\Pizza\backend
  npm run dev
  
  PowerShell #2 (Frontend):
  cd D:\Work\Pizza
  npm run dev


  КРОК 4: Оновити браузер
  ────────────────────────
  • Відкрийте http://localhost:5173
  • Натисніть Ctrl+Shift+R (жорстке оновлення)
